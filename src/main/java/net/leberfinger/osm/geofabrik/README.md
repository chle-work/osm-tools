# Geofabrik
Geofabrik has a great website that offers daily extracts for parts of OpenStreetMap's data, e.g. single continents or countries.

## Download URL Indexer
Unfortunately there is no index of all offered files other than the HTML pages. This package contains a tool to extract the links from the web page. The following chapters list the tool's output.
